Реализованные адаптивные функции

Гамбургер-меню для мобильных

На экранах меньше 1024px навигация скрывается и появляется кнопка-гамбургер. При клике открывается боковое меню с плавной анимацией.

@media (max-width: 1024px) { .nav-links { display: none !important; } .hamburger-btn { display: block; } }

Адаптивный Hero-заголовок

Заголовок использует clamp() и уменьшается до 32px на мобильных. Подзаголовок адаптируется до 16px.

.hero-title { font-size: clamp(44px, 5.5vw, 64px); } @media (max-width: 640px) { .hero-title { font-size: 32px !important; } .hero-subtitle { font-size: 16px; } }

Скрытие декоративных элементов

Карточка с демо-данными (hero-visual) и плавающие карточки скрываются на мобильных устройствах, освобождая место для контента.

@media (max-width: 640px) { .hero-visual, .float-card { display: none !important; } }

Вертикальные CTA-кнопки

На мобильных кнопки располагаются вертикально с полной шириной и минимальной высотой 48px для удобного касания.

@media (max-width: 640px) { .hero-ctas { flex-direction: column; gap: 12px; } .btn { width: 100%; min-height: 48px; } }

Коллапсируемые гриды

Features (3 колонки → 2 → 1), Pricing (2 → 1), How it works (3 → 1) — все секции адаптируются под размер экрана.

@media (max-width: 1024px) { .features-grid { grid-template-columns: repeat(2, 1fr); } } @media (max-width: 640px) { .features-grid, .how-grid, .pricing-grid { grid-template-columns: 1fr; } }

Адаптивные отступы

Padding секций уменьшается с 120px до 80px, боковые отступы с 48px до 20px. Навигация становится компактнее.

@media (max-width: 640px) { .landing-nav { padding: 12px 20px; } .hero { padding: 100px 20px 60px; } .features, .how, .pricing, .cta { padding: 80px 20px; } }
