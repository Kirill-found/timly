# Timly - Правила для Claude

## КРИТИЧЕСКИ ВАЖНО: БАЗА ДАННЫХ (ПРОДАКШН)

**Опасные команды (требуют подтверждения):**
- `DELETE FROM` - удаление данных
- `TRUNCATE` - очистка таблиц
- `DROP` - удаление таблиц/БД
- `UPDATE` - изменение данных
- `redis-cli FLUSHALL / DEL` - очистка кэша

**ОБЯЗАТЕЛЬНО перед выполнением:**
1. Спросить: "Это удалит данные. Ты уверен? (да/нет)"
2. Дождаться явного "да" или "удаляй" от пользователя
3. Только после подтверждения — выполнять

**Безопасные команды (можно без подтверждения):**
- `SELECT` - чтение данных
- `git pull`, `docker compose restart` - деплой
- Чтение логов

## Деплой

Безопасные команды:
- `git pull` - обновление кода
- `docker compose restart` - перезапуск сервисов
- `SELECT` запросы - только чтение
